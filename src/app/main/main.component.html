Secure Hash Algorithm (SHA) 512

March 2021

Utilized the BitVector module to create an implementation of SHA-512. The program reads in text in ASCII and outputs a file holding the secure hash corresponding to that input. To hash the input, it follows 4 steps: Pad the input message so that its length is an integer multiple of the block size, 1024 bits. Create a message schedule for the 512-bit input blocks being hashed. Carry out a round-based processing of a 1024-bit input message block (80 round in total). Lastly, mix and concatenate the contents of the hash buffer to obtain a 1024-element BitVector object.
